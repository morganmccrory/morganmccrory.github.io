CSS Design: Margin, Borders, and Padding

Let’s talk about the CSS design elements of margins, borders, and padding. These three elements control positioning in an HTML document. They allow for incredibly fine control over how our web page looks – down to the pixel. They do this by formatting the web page in what is called the CSS box model. Each HTML element acts like a box that holds the pictures and text that we code. This should make sense because even if we don’t format your HTML in any way, your text and pictures will still look like a box because of our web browser. It just won’t be very pretty!



The margin is the outermost element from our box. The larger the margin, the more space between our box and the boxes around it. We can adjust the margin to move our boxes (or elements) closer to or father away from each other. The syntax for margins is quite simple:

<margin: 15px;>


This creates a margin of 15 pixels on all four sides of our element. We can specify a margin to be on only one side of your element:

<margin-left: 20px;>


You can also set all of an element’s margins at once. Simply start with the top margin and go around clockwise, setting each side as you go. Here is what is should look like:

<margin: 1px 2px 3px 4px;>


This will set a top margin of 1 pixel, right of 2 pixels, bottom of 3 pixels, and left margin of 4 pixels. We can also use different measurements instead of pixels. Let’s move our margin 10% from the right.

<margin-right: 10%;>


Can you guess what would happen if we moved the margin -10px in a certain direction? The element would move the opposite direction!

The border is also exactly what it sounds like - the edge of our box. The border is invisible by default, but by setting the border property, we can make it visible. The syntax used to make the border visible is:

<border: 1px solid #d3d3d3;>


Can you guess what each part means? The border is 1 pixel wide, solid, and #d3d3d3 is the color code. (In this case, the color is gray. We can use either the name of a color or the color code if we would like to be more specific.) We can also make the border solid or dashed. And, just like with the margin, we can specify a border to be on only one side of our element.



Finally, we have the padding element. The padding is the space between the border and any other content that we want to include in your box. We adjust the padding to make the border either closer to or farther away from the other content. Pretty nifty! The syntax to adjust padding is:

<padding: 10px>


Just like with the margin and border, we can specify the padding to only be on one side of our element.

Putting It All Together

So what if we adjust the margin using pixels to center a box on a 13” screen and it looks good to us, but then someone else views our web page on a 15” screen and it is off-center for them? You guessed it – there is a solution! To center a box, you will simply input:

<margin-right: auto;>
<margin-left: auto;>


This centers the box no matter what screen size someone is viewing it on. I hope you learned a little in this introduction to CSS Design elements! Please check back soon!
